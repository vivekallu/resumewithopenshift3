SUDOKU SOLVER

Role : Used prolog's clpfd library to set constraints.

Tools Used : Prolog, clpfd library

  • Implementation: I created a sudoku predicate that defines constraints on the possible values (domain of the solution) that the solution can take. Solution is represented as list of lists (9 X 9). Each list (a row in sudoku) is contrained to have all different values and values can take 1 to 9.
  • I used transpose predicate to get the columns and set the same constraints on each column as row.
  • I created blocks predicate to create 3 X 3, lists and set the constraints that each block takes 1 to 9 (all different) and recursively call blocks for each 3 X 3 block of the whole matrix.
